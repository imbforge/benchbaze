# Subtree Source

This directory is maintained as a Git subtree in the main BenchBaze repository.

- Prefix: `collection/shared/map_dna/svg_converter`
- Upstream repository: `https://github.com/imbforge/benchbaze-map-dna-svg-converter.git`
- Upstream branch: `main`

## Pull updates

Run from repository root:

```bash
git subtree pull --prefix=collection/shared/map_dna/viewer https://github.com/imbforge/benchbaze-map-dna-svg-converter.git main
```
