# Subtree Source

This directory is maintained as a Git subtree in the main BenchBaze repository.

- Prefix: `collection/shared/map_dna/plannotate`
- Upstream repository: `https://github.com/imbforge/benchbaze-map-dna-plannotate.git`

## Pull updates

Run from repository root:

```bash
git subtree pull --prefix=collection/shared/map_dna/plannotate https://github.com/imbforge/benchbaze-map-dna-plannotate.git main
```
